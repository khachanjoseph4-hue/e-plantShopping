# e-plantShopping

This is the final project for the Coursera Front-End course.

## Project Description
This React app is a landing page for a fictional plant nursery called *Paradise Nursery*.

## Features
- A background image on the landing page
- A “Get Started” button
- Company details in the AboutUs section

## Project Files
- App.jsx – Displays company name and button
- App.css – Adds background image styling
- AboutUs.jsx – Contains company info
